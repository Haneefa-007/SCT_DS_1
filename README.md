# SCT_DS_1
Data Visualization Using Python
Project Overview

This project demonstrates basic data visualization techniques using Python. The goal is to visualize the distribution of categorical and continuous variables using:

Bar Charts
Histograms

The dataset contains sample information such as:

Name
Age
Gender

This project is beginner-friendly and introduces the fundamentals of:

Data handling
Visualization
Python libraries used in Data Science
Technologies Used
Python
Pandas
Matplotlib
VS Code / Jupyter Notebook
Libraries Required

Install the required libraries using:

pip install pandas matplotlib

The dataset used is a simple CSV file containing:

Name	Age	Gender

data.csv
